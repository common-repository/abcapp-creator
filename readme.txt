== Installation ==

1. Install directly ABCApp Creator through the plugin installer or upload the "abcapp-creator" folder to the "/wp-content/plugins/" directory
2. Activate the plugin by using the link provided by the plugin installer or through the "Plugins" menu in your WordPress.
